# DDNet Map Loader - Руководство пользователя

## Обзор
DDNet Map Loader - это инструмент, позволяющий быстро заменять карты на вашем сервере DDNet, сохраняя при этом позиции игроков с помощью функции горячей перезагрузки.

## Требования
1. Установленный сервер DDNet на вашем компьютере
2. Сервер настроен с паролем ECON (инструкции ниже)
3. Карты расположены в директории карт DDNet (инструкции ниже)

## Настройка папки с картами
1. Создайте директорию для карт, если она не существует:
   ```
   C:\Users\[ИмяПользователя]\AppData\Roaming\DDNet\maps
   ```
2. Скопируйте предоставленный файл `tutorial.map` из папки `maps` в:
   ```
   C:\Users\[ИмяПользователя]\AppData\Roaming\DDNet\maps\tutorial.map
   ```
   Это необходимо, так как это карта по умолчанию для сервера.

3. Любые дополнительные карты, которые вы хотите использовать, также должны быть помещены в эту директорию.

Примечание: Папка с картами пуста по умолчанию после установки DDNet. Без карты tutorial сервер и загрузчик карт не будут работать правильно.

## Настройка сервера
1. Перейдите в директорию сервера DDNet (например, `C:\Users\[ИмяПользователя]\Desktop\DDNet-18.7-win64\`)
2. Перейдите в папку `data`
3. Скопируйте предоставленный файл `myServerconfig.cfg` в эту папку
4. Запустите сервер с помощью команды:
   ```batch
   DDNet-Server.exe -f myServerconfig.cfg
   ```

### Детали конфигурации сервера
Предоставленный файл `myServerconfig.cfg` содержит:
```
sv_name "DDNet Map Loader Server"    # Имя сервера
sv_port 8303                         # Порт игры
sv_max_clients 16                    # Максимум игроков
sv_rcon_password "test123"          # Пароль администратора
sv_rcon_mod_password "test123"      # Пароль модератора
ec_port 8303                        # Порт внешней консоли
ec_password "test123"               # Пароль внешней консоли (используется Map Loader)
ec_bindaddr "127.0.0.1"            # Слушать только localhost
sv_map "tutorial"                   # Начальная карта (должна существовать в папке maps)
```

При изменении настроек убедитесь, что:
1. `ec_port` и `ec_password` соответствуют настройкам Map Loader
2. `ec_bindaddr` остается "127.0.0.1" для безопасности
3. Пароли изменены на более безопасные
4. `sv_map` указывает на существующую карту в вашей папке maps

## Установка
1. Поместите `ddnet_control.exe` в любую папку
2. Дважды щелкните по любому файлу `.map` для использования с загрузчиком

## Использование
### Метод 1: Двойной клик
1. Просто дважды щелкните по файлу `.map`, который хотите загрузить
2. Инструмент автоматически:
   - Создаст резервную копию текущей карты
   - Заменит текущую карту новой
   - Выполнит горячую перезагрузку сервера для сохранения позиций игроков

### Метод 2: Командная строка
```batch
ddnet_control.exe [путь_к_карте] [опции]
```
Опции:
- `--skip-backup`: Пропустить создание резервной копии
- `--restore`: Восстановить последнюю резервную копию вместо замены карты

## Структура директорий
Вот полная структура директорий, которая вам нужна:
```
C:\Users\[ИмяПользователя]\AppData\Roaming\DDNet\
└── maps\
    ├── tutorial.map      # Обязательная карта по умолчанию
    └── [ваши карты].map  # Ваши дополнительные карты

C:\Users\[ИмяПользователя]\Desktop\DDNet-18.7-win64\
└── data\
    └── myServerconfig.cfg    # Конфигурация сервера
```

## Устранение неполадок
1. Если программа не может подключиться к серверу:
   - Убедитесь, что сервер DDNet запущен с правильной конфигурацией
   - Проверьте соответствие пароля ECON в `myServerconfig.cfg`
   - Проверьте использование правильного порта (8303)

2. Если замена карты не удается:
   - Проверьте права доступа к директории карт
   - Убедитесь, что файл карты не используется другим процессом
   - Проверьте существование директории maps в `%AppData%\DDNet\maps`

3. Если сервер не запускается:
   - Убедитесь, что tutorial.map существует в вашей директории maps
   - Проверьте, что карта, указанная в `sv_map`, существует
   - Проверьте все пути и права доступа

## Заметки по безопасности
- Пароли по умолчанию в конфигурационном файле предназначены только для тестирования
- В рабочей среде измените все пароли на надежные уникальные значения
- Внешняя консоль привязана только к localhost для безопасности
- Делитесь конфигурацией сервера только с доверенными администраторами

## Примечания
- Программа автоматически создает резервные копии перед заменой карт
- Позиции игроков сохраняются при смене карт
- Сервер должен быть запущен для работы инструмента
- Всегда храните копию tutorial.map в вашей директории maps
